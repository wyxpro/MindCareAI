# 疗愈中心页面设计说明

## 设计概述
疗愈中心页面采用参考设计的优雅布局,以深色播放器为核心,配合呼吸动画、统计数据和冥想库列表,打造沉浸式的冥想体验。

## 设计参考
参考图片展示了现代化的冥想应用界面,具有以下特点:
- 简洁的顶部导航
- 圆角胶囊Tab切换
- 大卡片深色播放器
- 呼吸动画圆形
- 统计数据展示
- 冥想库列表

- **播放器**: slate-900 → slate-800 → slate-900
- **进度条背景**: slate-700
- **文字**: white / indigo-300

## 动画效果

### 1. 呼吸动画 (animate-breathe)
```css
@keyframes breathe {
  0%, 100% { transform: scale(1); opacity: 0.5; }
  50% { transform: scale(1.1); opacity: 1; }
}
```
- 应用于: 同心圆波纹
- 时长: 4秒
- 延迟: 0s / 0.5s / 1s

### 2. 脉冲发光 (animate-pulse-glow)
```css
@keyframes pulse-glow {
  0%, 100% { box-shadow: 0 0 20px rgba(99, 102, 241, 0.3); }
  50% { box-shadow: 0 0 40px rgba(99, 102, 241, 0.6); }
}
```
- 应用于: 中心圆形
- 时长: 2秒

### 3. 淡入上移 (animate-fade-in-up)
- 应用于: 所有卡片和列表项
- 递增延迟: 0.05s

### 4. 悬浮效果 (card-hover)
- 应用于: 统计卡片、冥想列表项
- 效果: 阴影加深 + 轻微上移

### 移动端 (< 768px)
- 单列布局
- 播放器圆形缩小 (w-48 h-48)
- 统计卡片保持3列
- Tab按钮自动换行

### 平板端 (768px - 1024px)
- 最大宽度: 2xl (672px)
- 播放器圆形标准尺寸 (w-64 h-64)

### 桌面端 (> 1024px)
- 最大宽度: 2xl (672px)
- 居中显示
